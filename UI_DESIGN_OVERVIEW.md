# Personal Finance Manager - UI Design Overview

## Design Philosophy

### Color Scheme
- **Primary**: Purple gradient (#667eea â†’ #764ba2) - Trust, sophistication
- **Success**: Green gradient (#28a745 â†’ #20c997) - Income, positive actions
- **Danger**: Red gradient (#dc3545 â†’ #e83e8c) - Expenses, alerts
- **Info**: Cyan gradient (#17a2b8 â†’ #0dcaf0) - Savings, goals
- **Warning**: Yellow gradient (#ffc107 â†’ #ff9800) - Budget warnings

### Typography
- **Font Family**: Segoe UI, system fonts
- **Headings**: 600 weight, professional
- **Body**: 400 weight, readable
- **Icons**: Bootstrap Icons (consistent style)

### Layout
- **Grid**: Bootstrap 5 responsive grid
- **Cards**: Rounded corners (10px), shadows, hover effects
- **Spacing**: Consistent margins and padding
- **Navigation**: Fixed top, gradient background
